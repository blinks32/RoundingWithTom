# Rounding with Tom – Decimal Rounding Educational Game

**Help Tom find his lost friends by rounding decimals to any place value!**

Fun math adventure built for **Legends of Learning** to teach rounding skills.

### Learning Objective
- Master **rounding decimals** to tenths, hundredths, or any specified place.
- Apply rounding rules in real-time decision-making scenarios.

### Gameplay
- Guide Tom through levels where correct rounding helps locate friends/pets.
- Progressive difficulty with immediate feedback on rounding choices.
- Engaging for upper elementary/middle school math standards.
